# Gray Matter

Gray Matter is a personal knowledge management tool designed to help you organize thoughts, web resources, and documents into a cohesive timeline. It's built for those who want to track how their understanding of a topic evolves over time.

## 🛠 What is this tool for?

The primary goal of Gray Matter is to prevent information from getting "lost" in bookmark folders or buried in note apps. Instead of just saving a link, you record your current perspective and confidence level on that piece of information.

### Core Workflow
1.  **Capture**: Add a web link, upload a document, or write a markdown note.
2.  **Organize**: Assign the entry to a **Topic**.
3.  **Reflect**: Add "Opinions" over time. As you learn more, your opinions change—Gray Matter tracks this evolution on a visual timeline.
4.  **Synthesize**: Use the Topic Synthesis view to look at all your resources and write a high-level summary of your current stance.

## 🚀 Key Features

*   **Unified Entry**: One place for links, PDFs/images, and notes.
*   **Layered Thinking**: Add multiple opinions to a single resource to see your growth.
*   **Topic Synthesis**: A dedicated space to combine various notes and sources into a final conclusion.
*   **Visual Evolution**: A pulse-animated timeline that maps out your thinking process.
*   **Classic Library**: A structured view of your topics using Roman numeral indexing.

## 🏗 Getting Started

### Prerequisites
- **Android Studio** (Hedgehog or newer)
- **Xcode 15+** (for the iOS app)

### Setup
1. Clone this repository (once uploaded).
2. Open the project in Android Studio.
3. To run on Android: Select the `androidApp` module and press `Shift + F10`.
4. To run on iOS: Open the `iosApp` folder in Xcode and hit the Run button.

## 🛠 Tech Stack
- **Framework**: Kotlin Multiplatform (KMP)
- **UI**: Jetpack Compose (Android) / SwiftUI (iOS)
- **Database**: SQLDelight (Local Storage)
- **Architecture**: MVVM with reactive Flows.
